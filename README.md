# eBird Rarity Mapper

## About the tool
This tool helps birders quickly explore notable eBird reports and visualize rarities on an interactive map.
It combines a static GitHub Pages frontend with a Cloudflare Worker backend proxy to the eBird API.

The app is hosted at: https://hydrospheric0.github.io/ebird-rarity-mapper/

## Features
- Interactive map with rarity markers and county overlays
- Region and species filtering controls
- Lower 48 rarity overlay (ABA code-based)
- Cloudflare Worker backend with CORS support for GitHub Pages
- Fast static frontend deployment on GitHub Pages

## How to use
1. Open the app in your browser.
2. Choose a region and (optionally) a species.
3. Adjust days-back and rarity options.
4. Toggle overlays to inspect notable reports and county-level rarity patterns.
5. Click map markers for observation details.
